利用 Hugo 與 Vercel 建構個人網頁

Apr 29, 2024
閱讀時長：1 分鐘

# Hugo

Hugo 是基於 Go 語言開發的靜態網站生成器，號稱為世界上最快的網站建構工具。除了生成網頁的速度極快，Hugo 也有許多主題可供選擇，並支援實時渲染、多語系切換的功能，可以優化文章的寫作與網站架設體驗。

# 安裝 Hugo

由於 Hugo 並未提供 GUI，因此需要由終端機安裝，以下是 Hugo 官方 提供在不同系統下的安裝方法：

# Windows
winget install Hugo.Hugo.Extended

# macOS
brew install hugo

# Linux
sudo snap install hugo

在安裝完成後使用以下指令測試 Hugo 是否安裝成功：

hugo version

# 安裝 Git

至 Git 官網下載安裝。

# 初始化網頁

選擇網頁存放的資料夾，並輸入以下指令新增網頁：

hugo new site myBlog

完成後會生成以下的檔案：

.
├── archetypes
│   └── default.md
├── assets
├── content
├── data
├── hugo.toml
├── i18n
├── layouts
├── static
└── themes

為了方便推送到 GitHub，我們先初始化整個 Git 儲存庫：

cd myBlog
git init

# 安裝主題

在 Hugo Themes 中有提供許多類型的主題，可以依個人喜好挑選，這裡我選擇 Stack 主題作為範例。

  1. 在網站的根目錄中打開終端機，用 git submodule 將主題下載：
git submodule add https://github.com/CaiJimmy/hugo-theme-stack/ themes/hugo-theme-stack
  1. 刪除 hugo.toml 並將 themes/hugo-theme-stack 中的 config.yaml 複製到根目錄中，並在檔案最上方加入以下內容：
baseurl: https://example.com
languageCode: zh-tw
theme: hugo-theme-stack
paginate: 5
title: myBlog
  1. 更多的設定（如：搜尋功能、側邊欄等）可以參考 stack 官方提供的 網頁配置。
